전체 데이터 흐름

사용자 입력, 병렬 분석, 저장소 조회, 산출물 생성이 한 화면에서 이어지도록 구성했습니다.

Frontend

Streamlit 홈, 분석 진행, 추천 결과, 상세 산출물 페이지가 Tier 1/2/3 점진 공개를 담당합니다.

  • 홈: 로그인·포트폴리오·자연어 질문
  • 결과: 한 줄 분석 신호와 근거 카드
  • 산출물: PDF·DOCX·Excel·HTML 다운로드

AI / LangGraph

Curator 이후 Quant, Qual, Competitor를 fan-out하고 Strategist와 Guardrail이 결론을 완성합니다.

  • Phase 1: mock 기반 E2E 파이프라인
  • Phase 2: RAG, Peer, 5y 밸류에이션 확장
  • 검증: Pydantic 스키마와 안전 표현 완화

DB / RAG

Postgres는 정형 데이터를, pgvector 테이블은 뉴스·공시 원문과 임베딩 청크를 관리합니다.

  • 정형: company, financial_data, stock_price
  • 사용자: users, holdings, analysis_history
  • RAG: rag_documents, rag_chunks
