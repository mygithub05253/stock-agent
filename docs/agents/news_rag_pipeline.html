뉴스 수집 및 RAG 임베딩 파이프라인

담당: 김도예 · 범위: 뉴스 수집, 전처리, pgvector 임베딩 적재, Qual Agent 검색 근거 제공

수집 소스
네이버 금융 종목 뉴스
저장 구조
raw_news → rag_documents → rag_chunks
임베딩 모델
BAAI/bge-m3 · 1024차원
검색 방식
Vector + Keyword Hybrid RRF

1. 전체 흐름

뉴스 파트는 단순 키워드 검색이 아니라, 수집된 뉴스를 RAG 문서로 정제하고 chunk 단위 임베딩을 생성해서 Qual Agent가 의미 기반 검색을 수행할 수 있게 만드는 역할입니다.

1

뉴스 크롤링

datas/news/news_collector.py가 company 테이블의 종목을 기준으로 네이버 금융 뉴스를 수집합니다.

2

원천 저장

뉴스 제목, 요약, URL, 발행일, 이벤트 타입, 감성 점수를 raw_news에 JSON payload로 저장합니다.

3

RAG 문서화

검색에 사용할 title + summary 기반 content를 rag_documents에 저장합니다.

4

Chunk + Embedding

datas/news/embed_news.py가 문서를 chunk로 나누고 bge-m3 임베딩을 생성합니다.

5

Qual 검색 근거

rag_chunks.embedding을 기반으로 Hybrid/RRF 검색 결과가 Qual evidence로 전달됩니다.

2. 테이블별 역할

테이블 역할 주요 필드
raw_news 수집 원본 보관. 재처리와 디버깅을 위한 source of truth. source, external_id, title, published_at, payload
rag_documents RAG 검색용 문서 단위 저장. 뉴스, 공시, 리포트를 같은 구조로 통합. source_type, corp_code, stock_code, content, metadata
rag_chunks 실제 검색 단위. chunk 본문과 pgvector 임베딩을 저장. document_id, chunk_index, content, embedding, search_tsv

3. 이번에 마무리한 부분

  • 완료rag_documents.id 기준으로 문서 조회하도록 수정
  • 완료뉴스 content를 500자 단위, 100자 overlap으로 chunking
  • 완료BAAI/bge-m3 임베딩 생성 후 rag_chunks.embedding 저장
  • 완료현재 스키마에 없는 published_at, stock_code, corp_code chunk 컬럼 사용 제거
  • 완료문서 업데이트 후 남는 stale chunk 삭제

4. 안정성 장치

  • 중복 방지ON CONFLICT (document_id, chunk_index, embedding_model)로 upsert
  • 추적성metadata에 source, external_id, title, embedding_model, embedder 기록
  • 검증DB 없이도 SQL 경로와 embedding 저장 여부를 테스트
  • pgvectorregister_vector(conn)로 vector 타입 저장 지원

5. 발표 때 말할 핵심

기존에는 뉴스가 rag_documents까지는 들어갔지만 rag_chunks.embedding 적재가 끊겨 있어서 의미 기반 검색이 실제 데이터에 적용되기 어려웠습니다. 지금은 뉴스 수집 후 chunk별 bge-m3 임베딩을 pgvector에 저장하므로, Qual Agent의 Hybrid/RRF 검색이 실제 임베딩 데이터를 사용할 수 있는 구조로 연결되었습니다.

6. 실행 순서

순서 명령 또는 파일 설명
1 python datas/news/news_collector.py 뉴스를 수집하고 raw_news와 rag_documents에 저장합니다.
2 python datas/news/embed_news.py rag_documents의 뉴스 문서를 chunking하고 rag_chunks에 embedding을 저장합니다.
3 src/stock_agent/rag/retriever.py Qual Agent가 query embedding과 keyword rank를 결합해 관련 뉴스를 조회합니다.
