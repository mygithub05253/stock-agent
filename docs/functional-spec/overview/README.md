# `docs/functional-spec/overview/` - 전체 기능 개요

> 기본·고급·시연 기능을 한 문서에서 탐색하기 위한 상위 명세 영역입니다.

## 파일

`functional_spec_all_features_v0.1.md`는 기능 ID, 사용자 가치, 입력·출력 관계를 전체 관점에서 정리합니다.

## 사용 원칙

- 개요는 탐색용이며 세부 계약은 `basic/`, `advanced/`, `demo/` 문서를 우선합니다.
- 실제 구현 상태는 [`IMPLEMENTATION_STATUS.md`](../IMPLEMENTATION_STATUS.md)를 우선합니다.
- 기능 추가 시 ID 충돌을 확인하고 세부 명세와 개요를 함께 갱신합니다.

[기능 명세 인덱스](../README.md)
