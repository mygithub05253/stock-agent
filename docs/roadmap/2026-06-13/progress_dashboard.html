집중 피드백 — 해소분과 남은 레버

강사 6대 피드백을 06-13 EOD 코드 기준으로 재분류했습니다. 초록 카드는 코드상 해소(재검증 대기), 빨강·주황 카드는 남은 작업입니다.

해소 Qual·Strategist 폴백 복원

qual.py는 retrieve_news_with_fallback/retrieve_disclosures_with_fallback로 retriever 실패를 try/except 처리하고, strategist.py는 일부 워커 결손을 허용(graceful degradation)합니다.

  • 해소 근거: #52(Qual fallback), #51(Strategist resilience)
  • 남은 일: CI 제외 테스트 재활성화 점검

해소 Guardrail 실게이팅

passed=True 하드코딩을 제거하고 7개 게이트(PII·욕설·보장표현·근거충분성 등) 기반 passed = not blocked + 재합성 rewrite 루프를 구현했습니다.

  • 해소 근거: #50(guardrail gating)
  • 남은 일: 관측(langsmith/langfuse) span 실연결

해소 Macro 파이프라인 연결

run_macro가 run_phase1_analysis에 포함되고, Strategist가 _WEIGHTS_WITH_MACRO(macro 0.15)로 결과를 실제 가중 소비합니다.

  • 해소 근거: #49(Macro 연결 + 가중치)
  • 남은 일: Macro 실데이터 폭 확대

해소 MCP 최소 실동작(#6)

Competitor DB 미연결 폴백이 mcp_bridge의 FastMCP 서버(sector_roster·market_metrics Tool)를 in-process 호출해 pykrx 실시간 시세 peer 비교를 확보합니다.

  • 해소 근거: #56(Competitor MCP). 루브릭 #6 0→~1
  • 남은 일: stdio 핸드셰이크 시연 캡처

P0 잔여 오픈웨이트 sLLM 실호출

config.py:34의 OpenRouter 기본 모델이 여전히 openai/gpt-4o-mini라 항목 #3(a) sLLM 요구를 미충족합니다. 가중 ×2 최대 레버.

  • 300B 이하 오픈웨이트(qwen-2.5-7b 등)를 1개 이상 노드에 실연결
  • 이미 구현된 Guardrail 게이팅·관측과 묶어 #3 회복
  • 담당: 백형준 중심, 이동원 설정·평가 지원

P0 잔여 LangGraph StateGraph 전환

pipeline.py는 여전히 8단계 동기 순차 호출입니다(StateGraph/add_node 0건, langgraph 의존성 없음). Multi-Agent 차별성(#1)이 가장 크게 막힌 지점, 가중 ×2.

  • worker fan-out(Send) + Strategist join + conditional 라우팅
  • 전환 시 #10 비동기 병렬도 동반 개선
  • 담당: 백형준, melinamuun 협업

P1 Hybrid + Reranker 실연결

retriever.py는 아직 pgvector 단독(<=>)입니다. DB의 tsvector GIN 인덱스는 준비됐으나 미사용.

  • plainto_tsquery 키워드 + 벡터 RRF 결합
  • bge-reranker + embed 단계 청킹 추가
  • 담당: doyekeem

P1 RAGAS 지표 확장 + 실데이터 재측정

RAGAS는 공식 4점이지만 context precision/recall과 채워진 DB 재실행 증거가 부족합니다.

  • context_precision·context_recall·relevancy 포함
  • Hybrid RAG 수정 후 누적 리포트 커밋
  • 담당: 이동원, doyekeem 협업
