# `docs/notion/images/` - Notion 문서 이미지

> `docs/notion/*.md`가 참조하는 로컬 시각 자료를 보관합니다.

현재 `image.png` 한 개가 있으며 Notion 원본의 맥락을 보존하는 용도입니다. 현재 README나 아키텍처의 공식 시각 자료는 `docs/assets/`를 사용합니다.

## 관리 규칙

- 원본 문서에서 실제 참조하는 파일만 유지합니다.
- 새 파일은 의미 있는 이름으로 저장합니다.
- 개인정보와 외부 서비스 비밀값이 보이는 캡처는 추가하지 않습니다.
