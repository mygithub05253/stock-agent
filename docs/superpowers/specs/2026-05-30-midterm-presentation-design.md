# 중간 발표 HTML 자료 설계 명세

## 개요

- **목적**: BDAI 12기 멀티에이전트 주식 분석 시스템 중간 발표용 HTML 슬라이드
- **발표 시간**: 10~15분 (Q&A 심화 대비 Appendix 포함)
- **대상 청중**: 혼합 (개발자 + 비개발자)
- **데모**: 별도 영상 파일 사용 (슬라이드 내 삽입 불필요)
- **구현 형식**: 단일 HTML 파일 (키보드/클릭 슬라이드 이동)

## HTML 구현 요건

### 네비게이션
- 키보드 화살표(←→) 및 클릭으로 슬라이드 이동
- 슬라이드 번호 표시 (현재/전체)
- Appendix는 메인 흐름과 분리 (별도 접근)

### 디자인 방향
- 다크 톤 배경 (전문성 강조)
- 섹션별 색상 구분
- 에이전트 플로우는 CSS/SVG 다이어그램으로 구현
- 모바일 고려 불필요 (발표용)

### 콘텐츠 원칙
- 슬라이드당 핵심 문장 1~3개 (발표자가 설명)
- 비개발자용 슬라이드: 비유 + 큰 숫자 강조
- 개발자용 슬라이드: 코드 스니펫 또는 구조 다이어그램 포함
