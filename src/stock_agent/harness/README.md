# `src/stock_agent/harness/` — 횡단(Cross-cutting) 컴포넌트

모든 에이전트가 *공통으로* 사용하는 안전·품질·해석 보조 모듈입니다.

## 파일

| 파일 | 역할 | 강의 연계 |
|------|------|-----------|
| `source_tracker.py` | 모든 수치/주장에 [출처] 강제 부착. 미부착 시 `ValueError` | W5 (책임 고지) |
| `glossary.py` | 한국어 금융용어 자동 풀이 (WACC ⓘ → "가중평균자본비용...") | UX 보조 |
| `disclaimer.py` | "교육용·투자권유 아님" 책임 고지 자동 부착 | W5 |
| `pii_filter.py` | 개인정보 마스킹 (이름·생년월일·계좌번호) | W5 |
| `sanitizer.py` | 크롤링 텍스트 욕설/광고/중복 정규화 | 강사 피드백 #1 |

## 작업 규칙

- 횡단 모듈은 **에이전트 코드를 직접 수정하지 않고** 데코레이터·미들웨어로 적용합니다.
- 가드레일이 차단한 모든 케이스는 `eval/reports/guardrail_blocks.jsonl` 에 자동 로깅합니다.
